# Module Template
This is a template project for creating new FieldKit module hardware in Kicad.

It Has : 
* The board outline.
* A connector to attach it to a FieldKit station.
* 3.3v Regulation.
* An EEPROM to identify the module and store any calibration data.

Clone and augment!
